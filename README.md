# Unieasy Explorer Hub

Campus discovery web app built with Vite + React + TypeScript.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

Requirements: Node.js (LTS) and npm.

```sh
npm install
npm run dev
```

## Scripts

```sh
npm run build
npm run preview
npm run lint
npm run test
```
